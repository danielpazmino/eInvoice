# eInvoice
Libreria para generar Facturas Electronicas (SRI)
## Información General
eInvoice es una libreria que permite generar, autorizar y enviar facturas electrónicas (SRI). Se la puede usar como una **Libreria (JAR)** o se puede deployar como un **Microservicio**. Actualmente permite generar los siguientes comprobantes electrónicos: 
* Facturas
* Retenciones
* Notas de Credito
### Desarrollado con
* Spring Boot
* Project Reactor
* JasperReport
* Librerias MITyCLib

### Caso de uso
La libreria es parte del sistema [SIF](http://sif.ec) que es una herramienta para la administracion de empresas
