# eInvoice
Libreria para generar Facturas Electronicas (SRI)
## Información General
eInvoice es una libreria que permite generar, autorizar y enviar facturas electrónicas (SRI). Se la puede usar como una **Libreria (JAR)** o se puede deployar como un **Microservicio**. Actualmente permite generar: 
* Facturas
* Retenciones
* Notas de Credito
### Construido con
* Spring Boot
* Project Reactor
* JasperReport
* Librerias MITyCLib
